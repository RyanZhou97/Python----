#Python选择与循环

##选择结构

###多分枝选择结构
	if 表达式 1:
    	语句块 1
	elif 表达式 2：
    	语句块 2
	elif 表达式 3:
    	语句块 3
	...
	else :
    	语句块 n
##循环结构

###带else的循环结构
>while循环和for循环都可以带else字句，如果循环因为条件表达式不成立而自然结束(不是因为执行了break而结束循环），则执行else结构中的语句,如果是因为break，则不执行

>迭代对象如果是空列表的话 也不会执行else

	while 条件表达式:
    	循环体
	else :
    	代码块


	for 取值 in 循环或迭代对象:
    	循环体
	else ：
    	代码块
